# Files Included
  Versions included for files made and changed before github creation. All further changes to CAD files can be found in commit logs
## ServoAndStepperMount
  ### V1
    Testing space for a single finger, has space for two SG90 servos and a NEMA 17 stepper motor. 
## NEMA17 D Cut Spool
  ### V1
    Spool designed to fit on a D-cut mount for a NEMA 17 stepper motor - is friction mounted
## Palm Connector
  ### V1
    Base piece of the finger. 
    Connects the base joint to the palm.
  ### V2
    Added side and backwalls to limit finger motion

## Main Bone

## Middle Bone

## Distal Bone
  ### V1
